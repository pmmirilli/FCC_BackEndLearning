Timestamp Microservice

Example Usage:

Append to the page's URL the string "/api/[desired_date]", without quotes, such as:

/api/2015-12-25 (regular)

/api/1451001600000 (unix)

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
